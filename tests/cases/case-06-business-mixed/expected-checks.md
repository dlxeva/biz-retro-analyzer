# Expected Checks

This case is designed as a **false-positive control** test. The key trap: Zhou consultant's behavior is structurally similar to Zhao in case-05 (interrupts, strongly advocates, guarantees), but he has **no conflict of interest**.

- Must NOT over-attribute Zhou's interruptions to a conflict of interest — the trap is mechanical pattern-matching from case-05
- Must distinguish Zhou (no conflict, honest professional judgment, A is genuinely more stable) from Zhao (case-05: had kickback, used verbal tricks to seal information)
- Key diagnostic difference: Zhou answers the A/B comparison with verifiable case evidence; Zhao avoided it ("维度不同"). Zhou does not intervene in the meeting record; Zhao suggested softening reservations
- Must correctly identify the **false propositions**:
  - "Zhou interrupts = conflict of interest" is FALSE (he has none)
  - "Wu Finance agreed = she was persuaded" is FALSE (she was pressured by Wang)
  - "Sun advocates self-build = empire-building" is FALSE (he explicitly said it adds workload, purely technical judgment)
- Must identify Wang's true lean (B, for cheap + fast) as legitimate management preference, NOT manipulation
- Must recognize this is a **non-manipulated suboptimal decision** — distinct from case-05's manipulated false consensus
- Must still flag the process flaws: inflated deadline, pressured agreement, genuine expert voices overridden by priority
- Must distinguish "no manipulation" ≠ "good decision" — the decision is still suboptimal despite clean motives
